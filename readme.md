# วิธีใช้งาน Drop2Me (P2P File Sharing)

โปรแกรม Drop2Me เป็นเว็บแอปพลิเคชันสำหรับส่งไฟล์หากันระหว่างเครื่องคอมพิวเตอร์ผ่านเครือข่ายโดยตรง (P2P) โดยไม่ต้องอัปโหลดไฟล์ไปฝากไว้ที่เซิร์ฟเวอร์

## สิ่งที่ต้องเตรียม (Prerequisites)

1.  **Python**: ต้องติดตั้ง Python ในเครื่องก่อน (แนะนำเวอร์ชัน 3.7 ขึ้นไป)
2.  **Library**: ต้องติดตั้งไลบรารี `aiohttp`

คำสั่งติดตั้งไลบรารี:
```bash
pip install aiohttp
```

## วิธีการรันโปรแกรม (Starting Server)

1.  เปิด Terminal หรือ Command Prompt ในโฟลเดอร์ที่เก็บไฟล์
2.  รันคำสั่ง:
    ```bash
    python server.py
    ```
3.  โปรแกรมจะแจ้งว่าเริ่มทำงานที่ `http://localhost:8080`

## วิธีใช้งาน (Usage)

ให้ผู้ใช้งานทั้ง 2 ฝ่าย เปิดเว็บเบราว์เซอร์แล้วเข้าไปที่ `http://localhost:8080` (หรือ IP ของเครื่องที่รัน server หากใช้งานคนละเครื่องกันในวง LAN)

### สำหรับผู้ส่ง (Sender)
1.  คลิกที่ปุ่ม **"สร้างห้องใหม่"** (Create Room)
2.  ระบบจะสร้าง **รหัสห้อง 6 หลัก** (เช่น `ABC123`) ขึ้นมา
3.  ส่งรหัสนี้ให้เพื่อนที่คุณต้องการส่งไฟล์ให้
4.  รอจนกว่าสถานะจะขึ้นว่า **"เชื่อมต่อแล้ว"** (Connected)
5.  ลากไฟล์ที่ต้องการส่งมาวางในกรอบ หรือคลิกเพื่อเลือกไฟล์
6.  คลิกปุ่ม **"ส่งไฟล์"**
7.  รอจนกว่าแถบความคืบหน้าจะเต็ม 100%

### สำหรับผู้รับ (Receiver)
1.  คลิกที่ปุ่ม **"เข้าร่วมห้อง"** (Join Room)
2.  กรอก **รหัสห้อง 6 หลัก** ที่ได้รับจากผู้ส่ง
3.  คลิกปุ่ม **"เข้าร่วม"**
4.  รอจนกว่าสถานะจะขึ้นว่า **"เชื่อมต่อแล้ว"** (Connected)
5.  เมื่อผู้ส่งเริ่มส่งไฟล์ จะมีแถบความคืบหน้าแสดงขึ้นมา
6.  เมื่อรับไฟล์เสร็จสิ้น ไฟล์จะถูกดาวน์โหลดลงเครื่องอัตโนมัติ

## หมายเหตุ
- หากใช้งานในวง LAN เดียวกัน ให้เปลี่ยน `localhost` เป็น IP Address ของเครื่องที่รัน Server (เช่น `192.168.1.100:8080`) เพื่อให้อีกเครื่องเข้าถึงได้
- การส่งไฟล์เป็นการส่งตรงระหว่างเครื่อง (P2P) ความเร็วขึ้นอยู่กับระบบเครือข่ายของทั้งสองฝ่าย
